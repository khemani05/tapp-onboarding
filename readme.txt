Plug-and-play onboarding for WooCommerce: Company → Department → Job Role with WP role mapping, guest purchase gating, admin CRUD & CSV, and My Account endpoint. No shortcodes required.

== Description ==
- Guests see price but cannot purchase (replace add-to-cart with Login to purchase, block cart/checkout).
- Registration adds Company, Department, Job Role, Phone (required) and Position (optional) on the same page with progressive reveal.
- Admin CRUD for Companies, Departments, Job Roles + CSV import/export and downloadable templates.
- My Account → Profile & Organization endpoint (edit Phone/Position; org assignment is admin-only).
- Roles provided: manager, staff, ceo (minimal caps).

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin.
3. Go to **TAPP Onboarding** menu to add Companies/Departments/Job Roles.
4. Test registration on **My Account** page.

== Frequently Asked Questions ==
= Does it require Elementor Pro? =
No, it’s hook-based and works with Elementor Free and WoodMart.

= Can guests see prices? =
Yes (configurable). They cannot purchase until logged in.
